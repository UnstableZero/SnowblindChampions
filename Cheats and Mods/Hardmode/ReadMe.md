The files in this folder are setup to be a drag-n-drop solution for playing Hardmode.
When you open the file in your browser, in the top right of the file display, there is a download raw file button.
Click the download raw file button, and drop the downloaded *.pnach file into your PCSX2 cheats folder.
In PCSX2, open the game properties for Champions: Return to Arms, and go to the cheats tab.
Enable Hardmode, and enjoy!
